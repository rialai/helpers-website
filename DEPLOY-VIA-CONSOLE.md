🚀 DEPLOYMENT ЧЕРЕЗ GOOGLE CLOUD CONSOLE
════════════════════════════════════════════════════════════════

ШАГИ:

1️⃣  ОТКРЫТЬ GOOGLE CLOUD CONSOLE
   → Зайти на: https://console.cloud.google.com
   → Выбрать проект: debraker-bot
   → В левом меню: Compute Engine → VM instances

2️⃣  НАЙТИ СЕРВЕР helpers.ie
   → Найти instance wordpress-gcp (или с IP 35.241.133.80)
   → Нажать на название instance

3️⃣  ОТКРЫТЬ ОБЛАЧНУЮ КОНСОЛЬ (SSH В БРАУЗЕРЕ)
   → Нажать кнопку "SSH" (будет в панели инструментов)
   → Откроется터минал в браузере

4️⃣  ВЫПОЛНИТЬ ДЕПЛОЙ
   
   Скопируй и вставь эти команды в консоль одну за другой:

   ────────────────────────────────────────────────────────────
   cd ~/bench/apps/helpers_website
   git fetch origin main
   git checkout main  
   git pull origin main
   cd ../..
   bench migrate
   bench clear-cache
   ────────────────────────────────────────────────────────────

5️⃣  ГОТОВО!
   → Заходи на https://helpers.ie
   → Ты должен увидеть новую интерактивную карту ресурсов
   → Она находится между секциями "Approach" и "What we do"

════════════════════════════════════════════════════════════════

ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

• Если ошибка "bench: command not found"
  → Может быть нужно: source ~/bench/env/bin/activate

• Если ошибка про .git
  → Проверь путь: ls -la ~, ls -la ~/bench/apps/

• Если git pull ничего не меняет
  → Может быть уже обновлено, проверь логи: git log --oneline -5
