Площадка для предпринимателей. Задумана для малого и среднего бизнеса (хендимэны), но будет незаменима так же и для предприимчивых менеджеров в крупных компаниях с одной стороны, а так же для людей которые по какой-то причине хоть и не занимаются ручным трудом но могут выполнять функциональный анализ для систем которые нужны для того, чтобы хендимен занимался исключительно хендименством.
